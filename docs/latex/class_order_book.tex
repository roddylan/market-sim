\doxysubsection{Detailed Description}
\doxylink{class_order}{Order} book storing market maker limit orders; handles match and make logic. 

Orders stored in ordered multiset. Buys sorted by best bid (highest) and earliest time. Sells sorted by best ask (lowest) and earliest time
